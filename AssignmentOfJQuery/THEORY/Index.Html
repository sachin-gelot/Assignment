Que-1. What is jQuery?

  • jQuery is a JavaScript library designed to simplify HTML DOM tree traversal and manipulation, as well as event handling, CSS animation, and Ajax. It is free, open-source software using the permissive MIT License.

Que-2. How to Apply CSS Using JQuery, How to Add Class and Remove Class in Jquery, JQuery Animation?

  1. we can use css() method to apple css in Jquery
    [ex- $("p").css("background-color", "yellow")]

  2. we can use addClass()method to add class in jquery
    [ex- $("button").click(function(){
    $("p:first").addClass("intro");
    });]

  3. we can use removeClass()method to remove class in jquery
    [ex- $("button").click(function(){
    $("p:first").removeClass("intro");
    });]

  4. The jQuery animate() method is used to create custom animations.
    [ex - $("button").click(function(){
    $("div").animate({left: '250px'});
    }); ]
